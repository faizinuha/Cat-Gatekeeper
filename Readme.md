# 🐱 Cat Gatekeeper

> Versi modifikasi tidak resmi dari ekstensi Chrome aslinya. Tidak dipublish di Chrome Web Store.

---

## Tentang

Cat Gatekeeper adalah ekstensi Chrome yang membantu kamu membatasi waktu penggunaan media sosial. Setelah batas waktu tercapai, seekor kucing lucu akan muncul dan menghalangi layar sampai waktu istirahat selesai.

---

## Fitur

- ⏱️ **Batas waktu** — atur berapa menit sebelum kucing muncul (1–480 menit)
- 😴 **Waktu istirahat** — atur durasi istirahat sebelum bisa lanjut (1–60 menit)
- 🌐 **All Sites** — aktifkan untuk memantau semua website, bukan hanya media sosial
- 📱 **Per-situs** — pilih situs mana saja yang ingin dipantau:
  - X (Twitter), YouTube, Facebook, Reddit, Threads, Bluesky
- 🐱 **Animasi kucing** — kucing bergerak lalu tertidur selama istirahat
- ❌ **Dismiss manual** — tutup kucing lebih awal lewat popup

---

## Cara Install (Manual / Developer Mode)

1. Download atau clone repo ini
2. Buka Chrome → pergi ke `chrome://extensions`
3. Aktifkan **Developer mode** (pojok kanan atas)
4. Klik **Load unpacked** → pilih folder ini
5. Ekstensi siap digunakan

---

## Cara Pakai

1. Klik ikon 🐱 di toolbar Chrome
2. Atur **Usage Limit** (menit sebelum kucing muncul)
3. Atur **Break Time** (menit istirahat)
4. Centang situs yang ingin dipantau, atau centang **AllSites** untuk semua web
5. Klik **Save**
